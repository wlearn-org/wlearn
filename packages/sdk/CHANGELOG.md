# Changelog

## 0.2.0

- Add `@wlearn/nn` exports: MLPClassifier, MLPRegressor, TabMClassifier, TabMRegressor, NAMClassifier, NAMRegressor
- Add `@wlearn/rf` exports: RFModel, loadRF
- Add `@wlearn/gam` exports: GAMModel, loadGAM
- Add `@wlearn/cluster` exports: ClusterModel, silhouette, calinskiHarabasz, daviesBouldin, adjustedRand, loadCluster
- Add `@wlearn/mitra` exports (optional peer dep): MitraClassifier, MitraRegressor, registerMitraLoaders
- SDK now re-exports 17 model packages (up from 12)
