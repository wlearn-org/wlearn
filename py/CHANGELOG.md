# Changelog

## 0.1.0

- Initial release
- Bundle encode/decode/validate (WLRN v1 format)
- Registry: global load() dispatcher with register()
- Pipeline: sequential estimator pipeline with save/load
- Preprocessing: StandardScaler, MinMaxScaler, Preprocessor
- AutoML: auto_fit with RandomSearch, SuccessiveHalvingSearch, PortfolioSearch, ProgressiveSearch
- Ensemble: VotingEnsemble, StackingEnsemble, BaggedEstimator, Caruana selection
- Model wrappers: XGBModel, LinearModel, SVMModel, KNNModel, LGBModel, EBMModel, BARTModel, TsetlinModel, XLearnModel
- Neural models: MLPClassifier, MLPRegressor, TabMClassifier, TabMRegressor, NAMClassifier, NAMRegressor
- Cross-language parity with JS @wlearn/* packages
- 517 tests
